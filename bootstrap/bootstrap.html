Hotel Booking Cancellation Prediction

An intelligent system built using machine learning to predict booking cancellations in hotels.

About This Project

Hello! I'm Sahil, a web and data science enthusiast. This dashboard is part of my college project to predict whether a hotel booking will be canceled using machine learning techniques.

The model is trained on real-world data from hotel bookings in Portugal and includes smart algorithms like Gradient Boosting and techniques such as SMOTE to handle data imbalance.
